BLUES WITH YOU

비 내리는 밤의 골목에서 빛과 장미의 흔적을 따라가는 3인칭 퍼즐 게임

▶ 바로 플레이하기 웹 브라우저에서 별도 설치 없이 실행할 수 있습니다.
게임 소개 게임 방법 실행 방법 링크

게임 소개

비가 그치지 않는 밤, 플레이어는 낯선 골목에서 한 사람의 흔적과 마주합니다.

여자의 모습은 직접 보여주지 않고, 골목에 남겨진 장미와 빛, 젖은 거리의 반사로 이야기를 전달합니다. 플레이어는 주변을 천천히 살피며 빛에 반응하는 오브젝트를 찾아야 합니다.

이 작품은 짧은 플레이 안에서 분위기와 상호작용을 경험할 수 있도록 구성한 프로토타입입니다.

게임 방법

목표

자판기에서 단서를 확인하고, 골목 끝의 포탈을 활성화해 닫힌 길을 통과하는 것이 목표입니다.

조작

입력기능
WASD플레이어 이동
마우스 이동카메라 회전
Left Shift달리기
E자판기·포탈 상호작용
마우스 오른쪽 버튼빛 오브젝트 타겟팅
마우스 왼쪽 버튼타겟 오브젝트 활성화
Esc마우스 커서 잠금 전환

진행 순서

  1. 게임 화면을 클릭해 브라우저 포커스를 맞춥니다.
  2. 골목을 이동해 자판기를 찾습니다.
  3. 자판기 화면이 화면 중앙에 오도록 바라본 뒤 E를 누릅니다.
  4. 자판기 화면의 빛 반응을 확인하고 골목 끝 포탈로 이동합니다.
  5. 포탈의 신호부를 중앙에 맞추고 E를 눌러 문을 엽니다.

종료 조건

포탈의 잠금이 해제되고 문이 올라가면 현재 프로토타입의 플레이 목표가 완료됩니다.

실행 방법

웹 실행

  1. 아래 플레이 링크를 엽니다.
  2. itch.io 페이지에서 플레이 버튼을 누릅니다.
  3. 게임 화면을 한 번 클릭한 뒤 입력을 시작합니다.

권장 환경: 최신 Chrome, Edge, Firefox 또는 Safari 데스크톱 브라우저

웹 버전은 브라우저에서 실행되므로 별도의 설치 파일이 필요하지 않습니다. 브라우저 탭이 키보드 입력을 받고 있는지 확인해 주세요.
